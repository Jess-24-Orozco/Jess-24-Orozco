## Portafolio de Ciencia de Datos

En esta sección presento algunos proyectos de **análisis de datos y machine learning** donde aplico limpieza de datos, análisis exploratorio, modelado predictivo y visualización para apoyar la **toma de decisiones basada en datos**.

| Proyecto | Problema | Solución / Modelo | Tecnologías | Resultados | Repositorio |
|----------|----------|------------------|-------------|------------|-------------|
| **Predicción de Cancelación de Clientes** | Identificar clientes con alta probabilidad de cancelar un servicio de telefonía. | Desarrollo de un modelo de **clasificación supervisada** para predecir el abandono de clientes. | Python, Pandas, Scikit-Learn | AUC-ROC **0.83** y Recall **0.80**, permitiendo detectar clientes con alto riesgo de cancelación. | 📂 [Ver Proyecto](LINK_REPO_CHURN) |
| **Predicción de Recuperación de Oro** | Estimar la eficiencia del proceso de recuperación de oro en una planta minera. | Construcción de modelos de **machine learning para regresión** utilizando datos industriales del proceso minero. | Python, Pandas, Scikit-Learn | Mejora en la estimación del rendimiento del proceso mediante modelado predictivo. | 📂 [Ver Proyecto](LINK_REPO_ORO) |
| **Análisis de Ventas de Videojuegos** | Comprender los patrones de ventas de videojuegos según región, plataforma y género. | **Análisis exploratorio de datos (EDA)** y visualización para identificar tendencias del mercado. | Python, Pandas, Seaborn | Identificación de plataformas y géneros con mayor rendimiento en diferentes mercados. | 📂 [Ver Proyecto](LINK_REPO_VIDEOJUEGOS) |
